飞龙招新啦

2019-09-04

飞龙创客俱乐部开始招新啦！

飞龙开发组简介：

飞龙开发组刚成立一年，指导老师是庞春江老师和王新颖老师，由夏天宇学长（16级）牵头，工作室位于教十楼A505。

工作室本次招新目标：

  1. 全部面向大二（18级）。
  2. 大概八人左右。
  3. 学习方向分为:
  • 前端： 暂时只是web前端，也即网页端。
  • 移动端： 暂时只是安卓端，java栈。
  • 后台： 有java栈 和 python栈。

工作室可以提供什么？

  1. 空调，不断电的那种。
  2. 一个比较好的学习氛围（不仅仅是课程上，更多是技术上）。
  3. 学长学姐帮助指导学习中遇到的各种问题。
  4. 千兆宽带了解一下（具体点就是下个东西20m/s左右）
  5. 热水，打印机。

我们需要什么样的同学？

百度某资深研发工程师在boss直聘上的动态是：“招最好的人，给最大的空间，看最后的结果，让优秀人才脱颖而出。”学习终归是自己的事。


我们希望你：

  1. 不要抱着水社团的心态，想着多一个比少一个好的报名。我们学习的东西是直接面向工作的，也即学到一个比较好的程度，是完全可以在本科就找到一个还不错的工作的。
  2. 学习这些需要大量的时间，这是可以拿去当吃饭的本事的，我们不希望你没有做好心里准备，或者没打算这样过大学的生活就报名。
  3. 如果你打算认真去争取保研，不打算投入时间在这些上面（当然这两者显然并不很冲突，只是个人的想法问题），但如果你学有余力，我们也很欢迎你。

统一回答一些可能存在的问题：

  1. 我们是学校老师带领成立的技术类工作室，全部成员均为华电二校在校生。目前工作室主要活跃人员有计算机系各个专业，跟即将到来的分流没有任何冲突。
  2. 预计可能会在中秋节放完回来之后再开始面试，大家可以趁这段时间看看自己喜欢什么方向（什么？到哪看？ bilibili是个神奇的地方，移动端搜 java 安卓，前端搜 html,css,js。后台搜 maven javaEE 或者 直接搜个 python flask 如果没有python基础，也可以先快速入个门 ）。
  3. 因为工作室初创，现在很自由，大家可以自行安排自己的时间，暂时不会有很紧迫的项目压力，也就是不会耽误到考前复习之类的。时间安排好的话不会影响到学习。
  4. 虽然没法做到项目驱动型学习，但是学习终归是自己的事，自己把基础打好，专业课学好，做点自己想做的东西。大三实习，等到大三那年暑假时，也未必在秋招上吃亏。
