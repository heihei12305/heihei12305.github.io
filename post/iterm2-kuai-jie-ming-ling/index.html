iTerm2 快捷命令

2020-01-14
  • command + t 新建标签🏷️
  • command + w 关闭标签🏷️
  • command + 数字 切换标签🏷️
  • command + 左右键 切换标签🏷️
  • command + enter 切换全屏
  • command + d 垂直分屏
  • command + shift + d 水平分屏
  • ctrl + u 清除当前行
  • ctrl + l 清屏
  • ctrl + a 到行首 ctrl + e 到行尾
