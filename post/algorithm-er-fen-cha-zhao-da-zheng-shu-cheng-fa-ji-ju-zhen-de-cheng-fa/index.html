1.二分查找


有序队列 a[0],a[1],a[2],....a[n]

注：可以重载一个"<",来表示其他的

  • a<b : a<b
  • a>b : a>b
  • a==b : !(a< b)&&!(b<a)
  • a<=b : !(b<a)

设a[-]的类型是T,有"<"

template<class T>
int bisearch(T *a,int l,int r,const T &v)
{
	if(r<l)
		return -1;
	//l<=r
	int m = ( r + l ) / 2
	if(v<a[m])
	{
		return bisearch(a,l,m-1,v);
	}
	if(a[m]>v)
	{
		return bisearch(a,m+1,r,v);
	}
	return m;
}

算法分析：

分+归并：工作量为0

最好情况：1次比较

最坏情况：O(logn)


2.大整数乘法

a * b = p*10^n + ( r - p - q)*10^(n/2) + q
