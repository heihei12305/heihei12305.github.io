docker

2019-06-17

docker的应用场景

  1. web应用的自动化打包和发布
  2. 自动化测试和持续集成，发布
  3. 在服务型环境中部署和调整数据库或其他的后台应用
  4. 从头编译或者扩展现有的OpenShift或Cloud Foundry平台来搭建自己的PaaS环境

docker的优点

  1. 简化程序
  2. 避免选择恐惧症
  3. 节省开支

docker使用

  • --help很有用，例如 docker stats --help
  • docker ps 来查看我们正在运行的容器
  • docker images 列出本地主机的镜像
  • docker search 查找镜像
  • docker pull 下载镜像
  • docker run 运行镜像
  • docker ps -a 查看以前的命令操作
  • docker rm 删除运行的镜像
  • docker run -itd -P --name mongo-server mongo bash 启动mogon服务
  • docker exec -it mongo-server bash 进入mongo容器
  • mongo & 启动mogon服务
  • ps -ef | grep mongo 在容器中查看mongo服务
