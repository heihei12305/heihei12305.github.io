restc 一个http服务器中间件

2019-06-23

restc官网链接


功能：restc可用于可视化显示请求，调试RESTful接口，搭载restc中间件后，访问者无需安装浏览器插件，直接通过浏览器访问接口地址，就可以可视化的方式对该接口发起各种请求。

  1. 可视化请求
  2. 调试RESTful接口


特色：

  1. 引入成本低，可以低成本的引入绝大多数现有的webAPI项目中。
  2. 使用方便
  3. 便于分享


指南


1.安装restc npm i restc
2.使用restc.expresss() 中间件

//import restc
const restc = require('restc');
//use restc middleware
app.use(restc.koa());
