阿里巴巴iconfont怎么是正确的使用方式

嘿嘿,自己操作记录一下嘿嘿

  1. 首先自己创建一个项目

  2. 然后找到自己喜欢的图片加入购物车

  1. 将购物车里的图片放到项目里

  1. 下载到本地 从 font class中 看到图标对应的class

  1. 本地解压使用， 需要先引入 iconfont.css

<link rel="stylesheet" href="./iconfont.css">

  1. 然后引用时 需要引入两个名字

<i class = "iconfont iconsearch"></i>

  1. 然后就ok了！

优点->免费，方便，可以通过css操作，也可以在iconfont存放图标的项目中直接修改
